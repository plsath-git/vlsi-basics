# VLSI Basics

A collection of small RTL examples demonstrating important digital design concepts, coding pitfalls, and corrected implementations.

## Current topics
- Combinational loop: bad vs good examples
- Procedural ordering: bad vs good examples

## Directory structure
- `rtl/combo_loop_bad/` — example with combinational loop issue
- `rtl/combo_loop_good/` — corrected combinational logic example
- `rtl/procedural_ordering_bad/` — problematic procedural ordering example
- `rtl/procedural_ordering_good/` — corrected procedural ordering example

## Goal
This repository is intended for learning, interview preparation, and illustrating RTL best practices.
